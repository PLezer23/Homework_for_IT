▌ Описание проекта

Данный скрипт предназначен для управления коллекцией книг в простом текстовом файле формата .txt. Скрипт позволяет добавлять, удалять, искать и просматривать книги, используя удобный интерфейс командной строки.

Скрипт хранит коллекцию книг в файле book.txt, где каждая книга имеет уникальный порядковый номер, год издания и название.

▌ Установка и запуск

Для запуска приложения выполните следующие шаги:

1. Скачайте исходный код на локальную машину.
2. Откройте терминал или консоль.
3. Перейдите в папку, содержащую файлы проекта.
4. Запустите скрипт командой:
   
Bash


   python main.py
   

▌ Использование

Программа выводит интерактивное меню с несколькими опциями. Чтобы выполнить определённое действие, выберите соответствующий пункт меню, введя нужный номер.

▌ Меню:

▌ 1. Добавление книги

Позволяет добавить новую книгу в базу данных. Вам потребуется ввести год издания и название книги.

▌ 2. Удаление книги

Можно удалить книгу по одному из трёх критериев: по номеру, по году издания или по названию. Выберите подходящий вам вариант и введите нужное значение.

▌ 3. Поиск книги

Позволяет искать книгу по номеру, году издания или названию. Введите интересующее значение и получите результат.

▌ 4. Просмотр всех книг

Показывает полный список всех книг, содержащихся в коллекции.

▌ 5. Полное удаление всех книг

Эта команда удаляет всю коллекцию книг из базы данных, очищая файл book.txt.

▌ 6. Выход

Завершение работы программы.

▌ 7. Замена (НЕ РЕАЛИЗОВАНО)

Пока эта функция не доступна. Она предназначена для возможности обновления данных существующей книги.

▌ Примечания

* Файл book.txt создаётся автоматически при первом запуске программы.
* Используйте команду выхода («6») для завершения работы программы.

▌ Техническая информация

Проект написан на языке программирования Python. Рекомендуется использовать версию Python 3.x.
